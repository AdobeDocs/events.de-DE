# Erweitern von AEM mit Adobe App Builder

Die Entwicklung von Microservices, die Integration von Backend-Systemen und die Automatisierung von Workflows war noch nie so einfach.

Seien Sie dabei, wenn Gastgeber Mark Szulc und Gäste wie Sarah Xu &amp; Zoran Nikolovski über reale Beispiele der Verwendung des neuen Adobe App Builder (ehemals Project Firefly) zur Erweiterung und Integration von Adobe Experience Manager diskutieren.  Von der Erweiterung vorhandener Funktionen in wenigen Minuten bis hin zur Erstellung neuer Funktionen werden Sie sehen, was inspiriert werden muss, und mit der Erstellung beginnen.
