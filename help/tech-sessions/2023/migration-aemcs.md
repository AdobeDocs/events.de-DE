# Migration zu AEM Cloud Service

In dieser Sitzung werden Ihnen die verfügbaren Techniken und Tools vorgestellt, um die Migration zu AEM Cloud Service zu vereinfachen. Nutzen Sie diese Aktivierungssitzung vom Content Transfer Tool bis zum neuen Cloud Acceleration Manager, der gemeinsam mit Best Practice Analyzer zur Bewertung Ihrer Umgebung verwendet wird, um sicherzustellen, dass Sie für eine erfolgreiche Live-Schaltung auf Cloud Service bereit sind.
