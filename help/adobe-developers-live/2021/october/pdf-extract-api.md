# Analysieren von PDF - Intelligentes Extrahieren von PDF-Inhalten in das JSON-Format mit der PDF Extract-API

Ein Blick darauf, wie der PDF-Extract-Service in der Adobe PDF Services-API das Analysieren von PDF so einfach machen kann wie das Analysieren von JSON. In dieser Sitzung erfahren Sie, was die PDF Extract-API tut, warum sie sich von allem unterscheidet, was Sie zuvor gesehen haben, und wie Sie sie verwenden können, um einzigartige PDF-Erlebnisse zu erstellen, ohne etwas über die PDF-Spezifikation oder PDF-„Interna“ zu wissen. Mit der PDF Extract API ist es nur JSON!
