* Eine Verbindung zwischen Autoren und Entwickler (Komponentenstruktur)
* Ein Debugging-Helper (Fehlerbenachrichtigung)
* Eine Testbegleitung (Tool zum Testen von Anwendungen) (Hobbes.js-AEM-Implementierung)
